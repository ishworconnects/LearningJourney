# Data Engineering Lab

This repo is where I keep my early data engineering practice. Most of the work
here lives in Jupyter notebooks, with a mix of class notes, short exercises,
and weekly assignments while I build up my Python basics.

## What's In Here

- `DE_Day2.ipynb` - introduction to variables, strings, and basic Python syntax
- `DE_Task_Day2.ipynb` - beginner list exercises and practice tasks
- `DE_Day3.ipynb` - working with `zip()`, tuples, lists, and paired data
- `DE_Day4.ipynb` - string methods and dictionary helper examples
- `DE_Dictionary.ipynb` - dictionary concepts and key-value practice
- `Task.ipynb` - short question-based exercises on dictionary usage
- `Week1_Assignment.ipynb` - week 1 assignment covering core Python concepts

## What I'm Practicing

- variables and data types
- strings and built-in string methods
- lists, tuples, sets, and dictionaries
- loops and simple data transformations
- beginner Python problem solving in notebooks

## Opening The Notebooks

Open the notebooks in Jupyter Notebook, JupyterLab, or VS Code.

```bash
jupyter notebook
```

Then open any notebook from the file browser and run the cells step by step.

I wanted one place to keep the work I am doing as I move from basic Python
